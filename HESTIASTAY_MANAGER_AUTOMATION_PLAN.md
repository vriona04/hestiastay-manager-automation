# 🏨 HestiaStay Manager Mobile Automation Framework

Mobile automation framework for the **HestiaStay Manager Android application** using **Python, Appium, Pytest, and Page Object Model**.

---

## Overview

This framework automates the HestiaStay Manager app, which is used by hostel owners/managers to manage hostel operations.

The framework supports:

- Real Android device automation
- Dashboard validation
- Bottom navigation testing
- Smoke testing
- Regression testing
- HTML report generation
- Screenshot/XML debugging
- GitHub version control

---

## Tech Stack

- Python 3.12
- Appium 3.5
- Pytest
- UiAutomator2
- Android Real Device
- Git & GitHub
